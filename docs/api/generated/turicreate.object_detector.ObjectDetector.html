turicreate.object_detector.ObjectDetector¶

class turicreate.object_detector.ObjectDetector(model_proxy=None, name=None)¶

A trained model using C++ implementation that is ready to use for classification or export to CoreML.

This model should not be constructed directly.

Methods

ObjectDetector.evaluate(dataset[, metric, …]) Evaluate the model by making predictions and comparing these to ground truth bounding box annotations.
ObjectDetector.export_coreml(filename[, …]) Save the model in Core ML format.
ObjectDetector.predict(dataset[, …]) Predict object instances in an SFrame of images.
ObjectDetector.save(location) Save the model.
ObjectDetector.summary([output]) Print a summary of the model.
