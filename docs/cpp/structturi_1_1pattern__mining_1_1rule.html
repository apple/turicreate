Detailed Description

rule structure Collection of information used for association rule mining and prediction.

A rule is (LHS -> RHS). Both LHS and RHS are frequent itemsets.

Vars: LHS (vector of size_t) - item ids in the current itemset RHS (vector of size_t) - item ids to add to the current itemset LHS_support (size_t) - support of LHS RHS_support (size_t) - support of RHS total_support (size_t) - support of LHS + RHS num_transactions (size_t) - normalizing constant for support values
